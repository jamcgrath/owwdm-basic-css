What Is Specificity?

  • A word I can't pronounce
  • How browsers decide which CSS property values are the most relevant to an element and, therefore, will be applied

How To Calculate Specificity

  • Give every inline style a value of 1000
  • Give every ID selector a value of 100
  • Give every class / attribute / pseudo-class selector a value of 10
  • Give every HTML/ pseudo-element selector a value of 1
  • Add them all up to get a specificity value.

Note: Universal selector (*), combinators (+, >, ~, ' ') and negation pseudo-class (:not) have no effect on specificity.
